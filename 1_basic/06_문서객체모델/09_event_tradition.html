이벤트(Event)

      이벤트(Event) : 동작, 행위
      -> 브라우저에서의 동작, 행위 : click, keyup, keydown

      이벤트 리스너
      -> 이벤트가 발생하는 것을 대기하고 있다가
         이벤트가 발생하는 것이 가지되면 연결된 기능(함수)를 수행하는 것

         (ex) onload, onclick, onkeyup, ...

      이벤트 핸들러
      -> 이벤트 리스너에 연결된 기능으로
         이벤트 발생 시 수행하고자하는 내용을 작성해둔 함수(function)

인라인 이벤트 모델

      요소 내부에 이벤트를 작성하는 방법
      on이벤트명 = "함수명()"

고전 이벤트 모델

      요소가 가지고 있는 이벤트 속성에 이벤트 핸들러를 연결하는 방법
      -> script에만 이벤트 관련 코드를 작성할 수 있다.

표준 이벤트 모델

      -w3c(HTML,CSS,JS 웹표준 재정 단체)에서
      공식적으로 지정한 이벤트 모델

      -한 요소에 여러 이벤트 핸들러 설정 가능
      -> 고전 이벤트 모델 단점 보완
